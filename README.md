### WordsFairyNoote词仙笔记（MVI+Kotlin+Compose+Hilt+Room）

# app介绍
一个UI十分漂亮的本地笔记app
项目中完全`kotlin`编写，采用学习的协程和MVI架构；适配深色模式等
采用了room 组件化、协程、flow等技术，适合初学者学习使用。此项目将长期维护，后续所有的新技术、平台适配等都将基于此项目实践。
